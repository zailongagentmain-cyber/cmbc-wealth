## 使用方法

```bash
cd ~/.openclaw/workspace/cmbc-wealth

# 同步产品列表
python3 scraper.py --products

# 同步详情 (净值+公告) - 慢慢跑
python3 scraper.py --details --limit 100

# 下载 PDF
python3 scraper.py --download

# 查看进度
tail -20 scraper.log
```
