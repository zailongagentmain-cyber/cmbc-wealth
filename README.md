# 📊 民生理财数据浏览器

基于 Streamlit 的民生理财产品数据浏览工具。

## 本地运行

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 在线部署

1. 点击按钮部署到 Streamlit Cloud：

[![Deploy to Streamlit](https://img.shields.io/badge/Deploy-Streamlit-FF4B4B?style=for-the-badge&logo=streamlit)](https://share.streamlit.io/deploy)

或者手动部署：
1. Fork 此仓库
2. 访问 https://share.streamlit.io
3. 关联 GitHub 账号，选择此仓库部署

## 数据

- 产品数量: 3615 只
- 净值记录: 18913 条
- 数据来源: 民生理财 (cmbcwm.com.cn)

## 功能

- 产品列表浏览
- 按状态、风险等级、业绩基准筛选
- 查看产品净值历史
