# PRD: 民生理财数据抓取与分析系统

> 针对民生理财 (cmbcwm.com.cn) 的理财产品数据进行抓取、存储与分析

## 1. 项目概述

- **项目名称**: CMBC Wealth Analyzer (民生理财分析器)
- **目标**: 自动抓取民生理财产品数据、净值、公告，构建查询与分析系统
- **数据源**: https://www.cmbcwm.com.cn
- **状态**: 开发中

## 2. 当前进度

### 技术验证

- SQLite 本地存储 ✅

- 数据解析 ✅

### 2.1 数据抓取

| 功能 | 优先级 | 说明 |
|------|--------|------|
| 产品列表 | P0 | 抓取全部 2286 只产品 |
| 产品详情 | P0 | 基本信息、销售文件 |
| 历史净值 | P0 | 每日净值数据 |
| 净值公告 PDF | P1 | 每日净值公告下载 |
| 产品公告 | P2 | 发行公告、重大事项等 |

### 2.2 数据存储

- SQLite 数据库
- 表: products, net_values, announcements

### 2.3 分析功能

- 净值走势分析
- 产品筛选 (按风险、收益率)
- 对比分析

## 3. 技术方案

### 3.1 爬虫方式

- 方式 A: 直接 API 调用 (需要分析 JavaScript)
- 方式 B: Browser 自动化 (Playwright/Selenium)
- 方式 C: 混合方案

### 3.2 数据源
